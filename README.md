# Mohamed Portfolio

Personal portfolio built with React, Vite, and React Three Fiber.

## Setup

```bash
pnpm install
pnpm dev
```

## Feedback Loop

```bash
pnpm format
pnpm format:check
pnpm lint
pnpm typecheck
pnpm test
pnpm check
```

The machine-readable feedback-loop contract lives in `.agents/feedback-loop.yml`.
Formatting is intentionally scoped to the maintenance surface: docs, config, contract files, and architecture tests.

Guardian note: the curated Guardian catalog available in this environment currently only contains Python CLI templates, so this repo uses an explicit repo-local overlay contract until a browser-first template is approved into the shared catalog.
